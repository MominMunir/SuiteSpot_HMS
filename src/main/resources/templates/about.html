About SuiteSpot

Hotel Management System

System Information

SuiteSpot is a comprehensive hotel management system designed to streamline operations and enhance guest experiences.

Key Features

  • Complete booking management system
  • Guest information and profile management
  • Room availability and status tracking
  • Check-in and check-out processing
  • Billing and invoice generation
  • Taxi request management
  • Admin dashboard with comprehensive reports
  • Database backup and restore functionality

Technology Stack

  • Backend: Spring Boot, Spring Data JPA, Spring Security
  • Database: PostgreSQL
  • Frontend: Thymeleaf, HTML5, CSS3
  • Build Tool: Maven
